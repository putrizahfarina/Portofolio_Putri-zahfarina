Project Saya

💻 Inventaris Barang

Project inventaris barang merupakan aplikasi peminjaman barang kampus dengan sistem yang digunakan untuk mencatat dan mengelola data barang yang tersedia di kampus. Aplikasi ini membantu mengatur proses peminjaman dan pengembalian barang oleh mahasiswa atau staf, serta memantau jumlah agar tetap terkontrol

🏣 Management gudang

Project ini merupakan Management gudang menggunakan kombinasi struktur data Hash Table dan AVL Tree untuk mengelola data barang secara efisien.
